README

There are four .py files in this submission. The indicators.py includes the fomulas for the indicators mentioned in this assignment and the figure generation functions. The TheoreticallyOptimalStrategy.py can be used for generating the trade dataframes for theoretically optimized situation and the bench mark situation. marketsimcode.py can be used to calculate the portfolio with the provided trade dataframe. And it will also calculate the cumulative return, std and mean of daily returns. 

testproject.py can be called to run all the python files mentioned above, and generate coresponding figures and results. Each individual file can be called separately as well.
